    02 变量概念及使用

    1、变量的基本概念

    动态语言的特点是交互性，会有数据的传递。而作为“中间人”需要进行数据的传递，传递的前提是PHP能自己存储数据（临时存储）。

    概念

    1、存储计算结果。

    2、变量是存在名字的。

    3、变量是可以通过名字访问

    4、变量是可以改变的：数据。

    2、变量的使用

    PHP中所有的所有变量名必须使用“$”符号

    1、定义：在系统中加入变量的名字（内存）

    2、赋值：可以将数据赋值给变量名（可在定义的同时完成）

    定义变量：在PHP中 不需要任何 关键字定义变量

    $var1; //定义变量

    $var2=1; //定义同时赋值

    3、可以通过变量名访问存储的数据。

    //访问变量

    echo $var2; //通过var2变量名字找到存储的内容1，然后输出

    4、可以将变量名从内存中删除

    //修改变量

    $var2=1;
$var2=3;
echo $var2;

    // 输出3

    //删除变量：使用unset（变量名）

    3、变量的命名规则

    1、在php中变量必须"$"符号开始；

    2、由字母、数字、_ 组成，不能数字开头；

    3、在PHP 中本身允许中文变量（不建议使用）

    4、预定义变量

    预定义：提前定义的变量 ，系统定义的变量、存储许多需要用到的数据（这些预定义变量都是数组）

    $_GET:所有表单以get方式提交的数据

$_POST: post提交的数据保存在此

$_REQUEST:get和post提交的数据都保存

$GLOBALS:PHP只所有的全局变量

$_SERVER:服务器信息

$_SESSION:session会话数据

$_COOKIE：cookie会话数据

$_ENV:环境信息

$_FILES:用户上传的文件信息

    5、可变变量

    可变变量：如果一个变量保存的值刚好是另外一个变量的名字，那么可以通过访问一个变量得到另一个变量的值：在变量前多加一个$符号。

    原理：先找到$a解析结果：b。将前面的$符号与结果b绑定，解析$b,得到$b的结果。

    6、变量传值

    将一个变量赋值给另外一个变量：变量传值

    变量传值方式：

    值传递：将变量保存的值复制一份，将新的值给另外一个变量保存（两个变量无关系）

    引用传递：将变量保存的值所在的内存地址，传递给另一个变量：两个变量指向同一块内存空间（两个变量是同一个值）

    $新变量=&$老变量

    在内存中，通常有以下几个分区

    栈区：程序可以操作的内存部分（不存数据，运行程序代码），少但是块

    代码段：存储程序的没存部分（不执行）

    数据段：存储普通数据（全局区和静态区）

    堆区：存储复杂数据，大但是效率低。
