    1、什么是C语言

    1.1 C语言是计算机语言

    计算机语言是任何计算机交流的语言 计算机语言：C C++ JAVA Python

    1.2 计算机语言的发展

    第一阶段

    机器语言：起初是原始的穿孔卡片，这种语言只有转接能理解

    ​ 二进制语言：

    ​ 正电代表 1 ；负电代表 0

    第二阶段

    汇编语言：用助计符来代替操作码，用地址符号或标号代替地址码。

    汇编语言被称为助计语言

    第三阶段

    高级语言：更接近于人类习惯的程序设计语言

    FORTRAN语言， COBOL语言，C语言，C++，C#， JAVA等等。

    1.3 C语言发展

    C语言-- >早期C语言不成熟--> 成熟 -->流行--> 国际标准ANSIC-->C89/C90--> 常见的C语言的编译器：Clang ,GCC ,WIN-TC.....

    4.2 变量的作用域

    局部变量：

    int main()
{
    {
        //局部变量的作用域只在局部变量，不能在全局生效
        int num=0;
    }
    printf("num=%d\n",num); //报错
    return 0;
}

    全局变量：

    int global =2020;           //全局变量的作用域是整个工程
void test()
{
    printf("test()-- %d\n",global);
}
int main()
    {
        test();                     //调用test函数
        printf("%d\n",global);
        return 0;
    }

    4.3生命周期

    局部变量的生命周期：是从{}局部变量开始到局部变量结束； 全局变量的生命周期：全局变量的周期和main函数的生命周期是一致，main函数生命周期是整个程序的生命周期；

    int main()
{
    {
        int a = 1;
        printf("%d\n",a);       //可以运行，a的生命周期在这个{}内；
    }
    printf("%d\n",a);           //error
    return 0;
}
